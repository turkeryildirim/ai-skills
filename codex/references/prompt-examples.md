# Prompt Formulation Examples

If you must formulate the prompt from a high-level goal, use these proven templates:

## 1. Pattern Detection
*Goal:* Find React hooks usage patterns.
*Prompt Content:* `Analyze this codebase and identify all React hooks usage patterns. Show how useState, useEffect, useContext, and custom hooks are being used. Provide file paths and brief snippets.`

## 2. Architecture Analysis
*Goal:* Architectural overview.
*Prompt Content:* `Analyze the overall architecture of this project. Identify main components, data flow, directory structure, and how different parts of the system interact. Highlight the entry points.`

## 3. Code Quality & Security
*Goal:* Find security issues.
*Prompt Content:* `Scan this codebase for potential security vulnerabilities, authentication/authorization issues, input validation problems, and unsafe data handling. List specific files and lines of concern.`

## 4. Feature Tracing
*Goal:* Trace feature implementation.
*Prompt Content:* `Trace the implementation of [feature] throughout the codebase. Show the complete execution path, including all files involved, data flow, and related API endpoints.`
